TP5420 Synchronous Boost Converter

2A Switch Current • 92% Efficiency • 1.2MHz Switching Frequency

Product Overview

The TP5420 is a high-efficiency synchronous step-up (boost) DC-DC converter that integrates two 100mΩ power MOSFETs and supports up to 2A switch current from a 2.5V to 14V input supply. The device operates at a fixed 1.2MHz switching frequency to allow for small external components and good efficiency across the entire load range.

The TP5420 offers a wide output voltage range from 3.3V to 28V with an adjustable version. Current mode control provides fast load transient response with excellent loop stability. Protection features include thermal shutdown, soft-start, cycle-by-cycle current limiting, and over-voltage protection.

Field Application Engineer Insights

When designing with the TP5420, select external components carefully to optimize performance. Use low ESR ceramic input and output capacitors for good ripple suppression. For the inductor, choose one rated for the saturation current at the maximum switch current limit.

The high switching frequency (1.2MHz) allows for small external components but requires careful PCB layout to minimize EMI. Use a solid ground plane and keep the power loop area as small as possible. The current mode control provides good stability but pay attention to slope compensation when operating at duty cycles greater than 50%.

Frequently Asked Questions

What is the difference between boost and flyback converters?

Boost converters are used for step-up applications where the output voltage is higher than the input voltage, and they use a single inductor for energy storage. Flyback converters can step-up or step-down and provide isolation between input and output, using a coupled inductor (transformer). Boost converters are simpler and more efficient for non-isolated applications.

How do I calculate the minimum inductance for the TP5420?

The minimum inductance can be calculated using the formula: Lmin = (VIN(min) × (VOUT - VIN(max))) / (ΔIL × fSW × VOUT). Where ΔIL is the inductor ripple current (typically 20-40% of the maximum output current) and fSW is the switching frequency. For the TP5420, use L ≥ 1.5μH for stable operation.

What is the purpose of the EN/UVLO pin?

The EN/UVLO pin serves dual purposes: it acts as an enable pin to turn the converter on/off, and it provides under-voltage lockout protection. The converter will automatically shut down if the input voltage drops below the UVLO threshold (typically 2.4V) and will restart when the input voltage rises above the rising threshold (typically 2.5V).
